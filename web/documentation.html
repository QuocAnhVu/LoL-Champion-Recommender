No really, what is this?

This Matchmaking Service is a collaborative filtering, machine learning recommender system. It is capable of assigning attributes to champions, and then it can tailor preferences to those attribute to each individual player.

How does it work?

First, Blitzcrank scrapes player mastery data to see which champions are usually played together by certain players. It uses these clusters to assign each champion a set of attributes. These attributes represent the facets of the game that players enjoy the most.

When a user enters their information into the web app, their own mastery data is scraped. The individual player's preferences are learned using their mastery data. The combination of individual preferences and champion attributes are combined to produce a single score for each champion. This score represents how much a player is likely to enjoy a champion.
